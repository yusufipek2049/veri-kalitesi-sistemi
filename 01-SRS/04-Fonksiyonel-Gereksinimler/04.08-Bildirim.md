# Uyarı ve Bildirim Yönetimi

### FR-059 — Eşik ve kritik başarısızlık bildirimi

| Alan | Tanım |
| --- | --- |
| Gereksinim ID | **FR-059** |
| Gereksinim adı | Eşik ve kritik başarısızlık bildirimi |
| Açıklama | Sistem skor eşik altına düştüğünde veya kritik kural başarısız olduğunda sistem içi bildirim oluşturmalıdır. |
| Aktör | Sistem |
| Öncelik | Must |
| Girdi | Skor olayı, alıcı, şablon |
| İşlem | Tetikleyici ve alıcı çözümleme |
| Çıktı | Notification kaydı |
| Ön koşullar | Sahiplik ve eşik mevcut |
| Son koşullar | Bildirim görünür |
| İş kuralları | RULE-006 |
| Hata durumları | Alıcı yok; şablon hatası |
| Kabul kriterleri | Olay tamamlandıktan sonra 5 dakika içinde bildirim kaydı oluşmalıdır. |
| İlgili kullanım senaryosu | UC-011, UC-012 |
| Bağımlılıklar | FR-028, FR-051 |
